# Time Series Analysis
## Overview
This project focuses on Analysing a tempreture datase to detect trends and seasonality.

The dataset contain Variables such as:
 - Date
 - Mean Tempreture
 - Humidity
 - Wind speed
 - Mean pressure

## Objectives
 - Plot time-series data and identify patterns.
 - Decompose the series into trend, seasonality, and esiduals using statsmodels.
 - Perform moving average smoothing and plot the results.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Statsmodels
- Jupyter Notebook
