# Data Cleaning and Preprocessing Project
## Overview
This Project focuses on cleaning and preprocessing a raw dataset using Python and Pandas.

The dataset contained:

- Missing values
- Duplicate records
- Iconsistent data formats
- Invalid data entry such as "Unknown" and "Error".

The goal was to prepare the dataset for further analysis 
by improving data quality and consistency.

## Objectives
- Load and inspect raw data
- Handle missing  values
- Remove duplicates
- Standardize inconsistency formats
- Convert incorect data types
- Prepare clean data for analysis
## Tools & Libraries used
 - Python
 - Pandas 
 - Jupyter Notebook
