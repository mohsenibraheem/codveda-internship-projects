# Codveda Data Analysis Internship Project
## Description:
This Repository contains all tasks completed during my Codveda Data Analysis internship programme.
The projects are organised into three levels, with each level focusing on different data analysis and machine learning concepts.

## Project Levels
### Level 1 Project
This level covers foundational data analysis tasks, including data cleaning, preprocessing and exploratory data analysis (EDA).

### Level 2 Project
This level focuses on intermediate data analysis and machine learning like regression analysis, time series analysis.

### Level 3 Projects
This projects contains advanced Machine Learning projects involving predicate modelling, model evaluation and optimisation techniques.

## Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn 
- Scikit-learn
- Jupyter Notebook
- Git

## How To Use
1. Clone the repository 
2. Open the desired project level folder
3. Read the README file inside that folder for detailed project information
4. Run the  notebooks in the Jupyter Notebook
